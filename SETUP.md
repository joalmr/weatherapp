# WeatherApp 🌦️
- Para la gestión de estados se aplica **Provider**
- El llamado a los servicio se realiza con **Dio**
- La navegación es manejada por **GoRouter**

## Estructura
Esta es la estructura de carpetas:

```css
lib/
├── core/
│   ├── constants/
│   ├── routes/
│   ├── theme/
│   └── utils/
├── features/
│   └── weather/
│       ├── data/
│       ├── domain/
│       └── presentation/
│── main.dart
│── .env
└── .env.dev
```
